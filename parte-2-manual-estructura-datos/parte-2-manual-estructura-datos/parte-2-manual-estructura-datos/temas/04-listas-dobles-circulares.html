Tema 4

Listas enlazadas dobles y circulares

Fundamentos, características, ventajas y aplicaciones de las listas enlazadas dobles y circulares.

Documentación

Definición de lista enlazada doble

Una lista enlazada doble es una estructura de datos lineal formada por nodos, donde cada nodo contiene tres partes: el dato, una referencia al nodo siguiente y una referencia al nodo anterior. Esto permite recorrer la lista en ambos sentidos: hacia adelante y hacia atrás.

Definición de lista enlazada circular

Una lista enlazada circular es una estructura de datos en la que el último nodo no apunta a null, sino al primer nodo de la lista. De esta forma, la estructura forma un ciclo continuo.

Características de la lista enlazada doble

  • Cada nodo tiene referencia al anterior y al siguiente.
  • Permite recorrido bidireccional.
  • Facilita inserciones y eliminaciones en ambos extremos.
  • Requiere más memoria que una lista enlazada simple.

Características de la lista enlazada circular

  • El último nodo apunta al primero.
  • No tiene un final nulo.
  • Permite recorridos repetitivos sin reiniciar la estructura.
  • Es útil cuando los datos se procesan en ciclos o turnos.

Ventajas de la lista doble

  • Se puede avanzar y retroceder.
  • Es más flexible para navegación en ambos sentidos.
  • Útil en historiales, reproductores y editores.

Ventajas de la lista circular

  • Evita volver a empezar la lista manualmente.
  • Muy útil en sistemas de turnos o procesos cíclicos.
  • Permite recorrer los elementos de manera continua.

Usos comunes

  • Navegación de historial.
  • Sistemas de reproducción de música o video.
  • Gestión de turnos (como en sistemas de atención al cliente).
  • Buffers circulares de memoria.

Ejemplo 1: Historial de acciones (Lista enlazada doble)

Descripción

Un editor de texto necesita almacenar las acciones realizadas por el usuario para permitir las opciones de "Deshacer" (ir al anterior) y "Rehacer" (ir al siguiente).

Pseudocódigo explicativo

INICIO

1. Crear una lista enlazada doble vacía.
2. Registrar varias acciones de forma secuencial.
3. Recorrer la lista hacia adelante para mostrar
   el historial en orden cronológico.
4. Recorrer la lista hacia atrás para simular
   el proceso de deshacer acciones.

FIN
Java 

public class Main {

    // Clase Nodo Doble
    static class NodoDoble {

        String accion;
        NodoDoble siguiente;
        NodoDoble anterior;

        NodoDoble(String accion) {

            this.accion = accion;
            this.siguiente = null;
            this.anterior = null;

        }

    }

    // Clase Lista Enlazada Doble
    static class ListaDoble {

        private NodoDoble cabeza;
        private NodoDoble cola;

        // Insertar al final
        public void registrarAccion(String accion) {

            NodoDoble nuevo =
                new NodoDoble(accion);

            if (cabeza == null) {

                cabeza = nuevo;
                cola = nuevo;

            } else {

                cola.siguiente = nuevo;
                nuevo.anterior = cola;
                cola = nuevo;

            }

        }

        // Recorrer hacia adelante
        public void recorrerAdelante() {

            NodoDoble actual = cabeza;
            int pos = 1;

            while (actual != null) {

                System.out.println(
                    pos + ". " + actual.accion
                );

                pos++;
                actual = actual.siguiente;

            }

        }

        // Recorrer hacia atrás
        public void recorrerAtras() {

            NodoDoble actual = cola;
            int pos = 1;

            while (actual != null) {

                System.out.println(
                    pos + ". " + actual.accion
                );

                pos++;
                actual = actual.anterior;

            }

        }

    }

    public static void main(String[] args) {

        ListaDoble historial =
            new ListaDoble();

        historial.registrarAccion(
            "Escribir texto"
        );

        historial.registrarAccion(
            "Cambiar color"
        );

        historial.registrarAccion(
            "Insertar imagen"
        );

        historial.registrarAccion(
            "Borrar párrafo"
        );

        System.out.println(
            "===== HISTORIAL DE ACCIONES (ADELANTE) ====="
        );

        historial.recorrerAdelante();

        System.out.println(
            "\n===== DESHACER ACCIONES (ATRÁS) ====="
        );

        historial.recorrerAtras();

    }

}

Ejemplo 2: Sistema de turnos (Lista enlazada circular)

Descripción

Un sistema de atención distribuye turnos entre varios asesores de forma continua. Cuando llega al último asesor, el siguiente turno vuelve a asignarse al primero.

Pseudocódigo explicativo

INICIO

1. Crear una lista enlazada circular
   con los asesores disponibles.
2. Enlazar el último nodo de vuelta
   con el primero para cerrar el ciclo.
3. Mostrar la lista de asesores
   en una vuelta completa.
4. Simular la asignación continua
   avanzando de forma cíclica.

FIN
Java 

public class Main {

    // Clase Nodo Circular
    static class NodoCircular {

        String asesor;
        NodoCircular siguiente;

        NodoCircular(String asesor) {

            this.asesor = asesor;
            this.siguiente = null;

        }

    }

    // Clase Lista Enlazada Circular
    static class ListaCircular {

        private NodoCircular cabeza = null;
        private NodoCircular cola = null;
        private NodoCircular actualPointer = null;

        // Agregar asesor a la lista circular
        public void agregarAsesor(String nombre) {

            NodoCircular nuevo =
                new NodoCircular(nombre);

            if (cabeza == null) {

                cabeza = nuevo;
                cola = nuevo;

                nuevo.siguiente = cabeza;

                actualPointer = cabeza;

            } else {

                cola.siguiente = nuevo;
                cola = nuevo;

                cola.siguiente = cabeza;

            }

        }

        // Contar elementos
        public int contar() {

            if (cabeza == null) {

                return 0;

            }

            int contador = 0;
            NodoCircular actual = cabeza;

            do {

                contador++;
                actual = actual.siguiente;

            } while (actual != cabeza);

            return contador;

        }

        // Recorrer una vuelta completa
        public void mostrarAsesores() {

            if (cabeza == null) {

                return;

            }

            NodoCircular actual = cabeza;
            int pos = 1;

            do {

                System.out.println(
                    pos + ". Asesor: " +
                    actual.asesor
                );

                pos++;
                actual = actual.siguiente;

            } while (actual != cabeza);

        }

        // Obtener el asesor actual
        // y avanzar al siguiente turno
        public String avanzarTurno() {

            if (actualPointer == null) {

                return "No hay asesores";

            }

            String asesorAsignado =
                actualPointer.asesor;

            actualPointer =
                actualPointer.siguiente;

            return asesorAsignado;

        }

        public String obtenerActual() {

            if (actualPointer == null) {

                return "Ninguno";

            }

            return actualPointer.asesor;

        }

    }

    public static void main(String[] args) {

        ListaCircular turnos =
            new ListaCircular();

        turnos.agregarAsesor(
            "Asesor Juan"
        );

        turnos.agregarAsesor(
            "Asesor Ana"
        );

        turnos.agregarAsesor(
            "Asesor Luis"
        );

        turnos.agregarAsesor(
            "Asesor María"
        );

        System.out.println(
            "===== SISTEMA DE TURNOS CIRCULAR ====="
        );

        System.out.println(
            "Cantidad de turnos: " +
            turnos.contar()
        );

        System.out.println(
            "Listado de turnos:"
        );

        turnos.mostrarAsesores();

        System.out.println(
            "--------------------------------------"
        );

        System.out.println(
            "Turno actual: " +
            turnos.obtenerActual()
        );

        // Simulación de atención continua
        System.out.println(
            "Atendiendo en: " +
            turnos.avanzarTurno()
        );

        System.out.println(
            "Atendiendo en: " +
            turnos.avanzarTurno()
        );

        System.out.println(
            "Siguiente turno: " +
            turnos.obtenerActual()
        );

    }

}

Retos

Reto 1: Historial de navegación

Un navegador guarda las páginas visitadas en una lista enlazada doble. El sistema debe mostrar el historial en orden normal y luego en orden inverso.

Pistas

  • Usa una lista doble para poder moverte en ambos sentidos.
  • Guarda las páginas en el orden en que fueron visitadas.
  • Recorre desde la cabeza hasta la cola y luego al revés.
  • La última página visitada debe aparecer primero en el recorrido inverso.
Salida esperada 
===== HISTORIAL DE NAVEGACIÓN =====

Páginas visitadas:

1. Inicio
2. Productos
3. Carrito
4. Pago

Recorrido inverso:

5. Pago
6. Carrito
7. Productos
8. Inicio

Reto 2: Rotación de tareas en una lista circular

Un equipo de trabajo utiliza una lista circular para asignar tareas por turnos. El sistema debe mostrar las tareas en una vuelta completa y luego avanzar al siguiente turno.

Pistas

  • Usa una estructura circular para evitar finalizar la lista.
  • El último nodo debe apuntar al primero.
  • Muestra los elementos una sola vuelta completa.
