Tema 7

Tablas Hash

Estructuras de acceso rápido mediante pares clave-valor y funciones hash.

Documentación

Definición de tabla hash

Una tabla hash es una estructura de datos que almacena pares clave-valor, donde cada clave se transforma mediante una función hash en una posición dentro de una tabla.

Esto permite acceder, insertar y eliminar datos de forma muy rápida, generalmente en tiempo constante O(1).

Función hash

Es un algoritmo que convierte una clave en un índice numérico dentro del arreglo.

Una buena función hash distribuye los datos de manera uniforme para evitar colisiones.

Colisiones

Ocurren cuando dos claves diferentes generan el mismo índice.

Existen técnicas para resolverlas:

  • Encadenamiento (chaining): Cada posición almacena una lista de elementos.
  • Direccionamiento abierto: Se busca otra posición disponible dentro de la tabla.

Operaciones básicas de una tabla hash

  • put (insertar): Agrega un par clave-valor.
  • get (obtener): Recupera el valor asociado a una clave.
  • remove (eliminar): Elimina un elemento por su clave.
  • containsKey: Verifica si una clave existe.

Características de la tabla hash

  • Acceso rápido a los datos.
  • No mantiene orden de los elementos.
  • Depende de una buena función hash.
  • Puede manejar colisiones.

Ventajas de la tabla hash

  • Búsqueda, inserción y eliminación muy rápidas.
  • Ideal para grandes volúmenes de datos.
  • Eficiente en comparación con listas o arreglos para búsquedas.

Desventajas de la tabla hash

  • Puede tener colisiones.
  • No mantiene orden.
  • Puede consumir más memoria.

Usos comunes

  • Bases de datos (índices).
  • Sistemas de autenticación (usuarios y contraseñas).
  • Diccionarios y mapas.
  • Cachés de datos.
  • Conteo de frecuencias.

Ejemplo 1: Registro de estudiantes con clave

Descripción

Un sistema académico guarda estudiantes usando un código único como clave y su nombre como valor.

Luego permite consultar un estudiante por su código.

Pseudocódigo explicativo 

INICIO

1. Crear una tabla hash vacía.
2. Insertar varios estudiantes (clave, nombre).
3. Buscar un estudiante por su clave.
4. Mostrar el resultado.

FIN.
Java 

import java.util.HashMap;

public class Main {

    public static void main(String[] args) {

        HashMap<String, String> estudiantes =
            new HashMap<>();

        estudiantes.put("E01", "Carlos");
        estudiantes.put("E02", "María");
        estudiantes.put("E03", "Juan");
        estudiantes.put("E04", "Laura");

        System.out.println(
            "===== REGISTRO DE ESTUDIANTES ====="
        );

        String codigo = "E03";

        System.out.println(
            "Buscando estudiante con código "
            + codigo
        );

        if (estudiantes.containsKey(codigo)) {

            System.out.println(
                "Estudiante: "
                + estudiantes.get(codigo)
            );

        } else {

            System.out.println(
                "Estudiante no encontrado."
            );

        }

    }

}

Ejemplo 2: Conteo de palabras

Descripción

Un programa cuenta cuántas veces aparece cada palabra en un texto utilizando una tabla hash.

Pseudocódigo explicativo 

INICIO

1. Crear una tabla hash vacía.
2. Leer un texto.
3. Separar el texto en palabras.

4. Para cada palabra:

   a. Si existe, incrementar contador.
   b. Si no existe, agregar con valor 1.

5. Mostrar resultados.

FIN.
Java 

import java.util.HashMap;

public class Main {

    public static void main(String[] args) {

        String texto =
            "hola mundo hola java mundo hola";

        HashMap<String, Integer> conteo =
            new HashMap<>();

        String[] palabras = texto.split(" ");

        for (String palabra : palabras) {

            if (conteo.containsKey(palabra)) {

                conteo.put(
                    palabra,
                    conteo.get(palabra) + 1
                );

            } else {

                conteo.put(palabra, 1);

            }

        }

        System.out.println(
            "===== CONTEO DE PALABRAS ====="
        );

        for (String palabra : conteo.keySet()) {

            System.out.println(
                palabra + ": "
                + conteo.get(palabra)
            );

        }

    }

}
