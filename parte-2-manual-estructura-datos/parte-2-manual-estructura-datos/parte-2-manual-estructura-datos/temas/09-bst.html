Árboles binarios de búsqueda BST

Árboles binarios de búsqueda, operaciones, características, ventajas, desventajas y aplicaciones.

Documentación

Definición de árbol binario de búsqueda (BST)

Un árbol binario de búsqueda (BST, Binary Search Tree) es un tipo especial de árbol binario que mantiene un orden específico: Todos los valores del subárbol izquierdo son menores que el nodo raíz, y todos los valores del subárbol derecho son mayores. Esta propiedad permite realizar búsquedas, inserciones y eliminaciones de manera eficiente.

Propiedad fundamental del BST

  • Izquierda < Raíz < Derecha
  • Se cumple recursivamente en todo el árbol.

Operaciones básicas de un BST

  • Insertar: Ubica el valor en la posición correcta según su valor.
  • Buscar: Compara el valor y decide si ir a la izquierda o derecha.
  • Eliminar: Quita un nodo considerando tres casos: Nodo hoja, Nodo con un hijo o Nodo con dos hijos.

Características del BST

  • Mantiene los datos ordenados.
  • Permite búsquedas eficientes.
  • Es una estructura dinámica.
  • Puede desbalancearse si los datos se insertan en orden.

Ventajas del BST

  • Búsqueda rápida ($O(\log n)$ en promedio).
  • Inserción y eliminación eficientes.
  • Permite recorridos ordenados (inorden).

Desventajas del BST

  • Puede volverse lento $(O(n))$ si está desbalanceado.
  • Requiere más lógica que estructuras simples.

Usos comunes

  • Bases de datos.
  • Sistemas de búsqueda.
  • Índices.
  • Estructuras jerárquicas ordenadas.

Ejemplo 1: Inserción en un árbol binario de búsqueda

Descripción

Un sistema inserta varios valores en un BST y luego muestra el recorrido en inorden (ordenado).

Pseudocódigo explicativo

INICIO

1. Crear un árbol vacío.

2. Insertar valores:
a. Si el valor es menor, ir a la izquierda.
b. Si es mayor, ir a la derecha.

3. Recorrer en inorden.

4. Mostrar los valores ordenados.
FIN
Java 

public class Main {
    static class Nodo {
        int dato;
        Nodo izquierdo;
        Nodo derecho;

        Nodo(int dato) {
            this.dato = dato;
        }
    }

    // Insertar en BST
    public static Nodo insertar(Nodo raiz, int dato) {
        if (raiz == null) {
            return new Nodo(dato);
        }
        if (dato < raiz.dato) {
            raiz.izquierdo = insertar(raiz.izquierdo, dato);
        } else {
            raiz.derecho = insertar(raiz.derecho, dato);
        }
        return raiz;
    }

    // Recorrido inorden
    public static void inorden(Nodo nodo) {
        if (nodo != null) {
            inorden(nodo.izquierdo);
            System.out.print(nodo.dato + " ");
            inorden(nodo.derecho);
        }
    }

    public static void main(String[] args) {
        Nodo raiz = null;
        raiz = insertar(raiz, 10);
        insertar(raiz, 5);
        insertar(raiz, 15);
        insertar(raiz, 3);
        insertar(raiz, 7);

        System.out.println("===== ÁRBOL BINARIO DE BÚSQUEDA =====");
        System.out.print("Recorrido en inorden: ");
        inorden(raiz);
    }
}
Salida esperada 
===== ÁRBOL BINARIO DE BÚSQUEDA =====
Recorrido en inorden: 3 5 7 10 15

Ejemplo 2: Eliminación de un nodo en un BST

Descripción

Un sistema elimina un valor de un árbol binario de búsqueda considerando los diferentes casos posibles.

Pseudocódigo explicativo

INICIO

1. Buscar el nodo a eliminar.

2. Si es hoja, eliminar directamente.

3. Si tiene un hijo, reemplazar por el hijo.

4. Si tiene dos hijos:
a. Buscar el sucesor (mínimo del subárbol derecho).
b. Reemplazar el valor.
c. Eliminar el sucesor.

5. Mostrar el árbol actualizado.
FIN
Java 

public class Main {
    static class Nodo {
        int dato;
        Nodo izquierdo;
        Nodo derecho;

        Nodo(int dato) {
            this.dato = dato;
        }
    }

    public static Nodo insertar(Nodo raiz, int dato) {
        if (raiz == null) return new Nodo(dato);
        if (dato < raiz.dato) {
            raiz.izquierdo = insertar(raiz.izquierdo, dato);
        } else {
            raiz.derecho = insertar(raiz.derecho, dato);
        }
        return raiz;
    }

    // Encontrar el mínimo
    public static Nodo minimo(Nodo nodo) {
        while (nodo.izquierdo != null) {
            nodo = nodo.izquierdo;
        }
        return nodo;
    }

    // Eliminar nodo
    public static Nodo eliminar(Nodo raiz, int valor) {
        if (raiz == null) return null;

        if (valor < raiz.dato) {
            raiz.izquierdo = eliminar(raiz.izquierdo, valor);
        } else if (valor > raiz.dato) {
            raiz.derecho = eliminar(raiz.derecho, valor);
        } else {
            if (raiz.izquierdo == null && raiz.derecho == null) {
                return null;
            }
            if (raiz.izquierdo == null) return raiz.derecho;
            if (raiz.derecho == null) return raiz.izquierdo;

            Nodo sucesor = minimo(raiz.derecho);
            raiz.dato = sucesor.dato;
            raiz.derecho = eliminar(raiz.derecho, sucesor.dato);
        }
        return raiz;
    }

    public static void inorden(Nodo nodo) {
        if (nodo != null) {
            inorden(nodo.izquierdo);
            System.out.print(nodo.dato + " ");
            inorden(nodo.derecho);
        }
    }

    public static void main(String[] args) {
        Nodo raiz = null;
        raiz = insertar(raiz, 10);
        insertar(raiz, 5);
        insertar(raiz, 15);
        insertar(raiz, 3);
        insertar(raiz, 7);

        System.out.println("===== ELIMINACIÓN EN BST =====");
        System.out.print("Árbol original: ");
        inorden(raiz);
        System.out.println();

        raiz = eliminar(raiz, 5);
        System.out.print("Árbol después de eliminar 5: ");
        inorden(raiz);
    }
}
Salida esperada 
===== ELIMINACIÓN EN BST =====
Árbol original: 3 5 7 10 15
Árbol después de eliminar 5: 3 7 10 15
