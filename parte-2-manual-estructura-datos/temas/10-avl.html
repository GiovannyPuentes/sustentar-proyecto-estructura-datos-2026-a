Árboles balanceados — AVL

Árboles AVL, balanceo automático, rotaciones, operaciones y aplicaciones.

Documentación

Definición de árbol AVL

Un árbol AVL es un árbol binario de búsqueda (BST) auto-balanceado, donde la diferencia de altura entre el subárbol izquierdo y derecho de cualquier nodo no puede ser mayor a 1. Esto garantiza que el árbol siempre se mantenga equilibrado.

Propiedad fundamental del AVL

  • Es un BST: Izquierda < Raíz < Derecha.
  • Se mantiene balanceado automáticamente.
  • Factor de balance ∈ {-1, 0, 1}.

Operaciones básicas de un AVL

  • Insertar: Inserta como BST y luego balancea.
  • Eliminar: Elimina como BST y luego balancea.
  • Rotar: Corrige desbalances (LL, RR, LR, RL).

Características del AVL

  • Altura balanceada.
  • Búsqueda eficiente garantizada.
  • Uso de rotaciones.
  • Más estricto que BST.

Ventajas del AVL

  • Siempre O(log n).
  • Búsquedas muy rápidas.
  • No se desbalancea.

Desventajas del AVL

  • Implementación más compleja.
  • Más operaciones (rotaciones).
  • Inserciones más costosas que BST.

Usos comunes

  • Sistemas de bases de datos.
  • Índices en memoria.
  • Motores de búsqueda.
  • Sistemas que requieren datos ordenados rápidos.

Ejemplo 1: Inserción en un árbol AVL

Descripción

Un sistema inserta valores en un árbol AVL y automáticamente realiza rotaciones para mantener el equilibrio.

Pseudocódigo explicativo

INICIO

1. Insertar como en BST.

2. Calcular altura del nodo.

3. Obtener factor de balance.

4. Si está desbalanceado:
a. Aplicar rotación correspondiente (LL, RR, LR, RL).

5. Mostrar el árbol balanceado.
FIN
Java 

public class Main {
    static class Nodo {
        int dato;
        int altura;
        Nodo izquierdo, derecho;

        Nodo(int dato) {
            this.dato = dato;
            this.altura = 1;
        }
    }

    static int altura(Nodo n) {
        if (n == null) return 0;
        return n.altura;
    }

    static int balance(Nodo n) {
        if (n == null) return 0;
        return altura(n.izquierdo) - altura(n.derecho);
    }

    static Nodo rotarDerecha(Nodo y) {
        Nodo x = y.izquierdo;
        Nodo T2 = x.derecho;

        x.derecho = y;
        y.izquierdo = T2;

        y.altura = Math.max(altura(y.izquierdo), altura(y.derecho)) + 1;
        x.altura = Math.max(altura(x.izquierdo), altura(x.derecho)) + 1;

        return x;
    }

    static Nodo rotarIzquierda(Nodo x) {
        Nodo y = x.derecho;
        Nodo T2 = y.izquierdo;

        y.izquierdo = x;
        x.derecho = T2;

        x.altura = Math.max(altura(x.izquierdo), altura(x.derecho)) + 1;
        y.altura = Math.max(altura(y.izquierdo), altura(y.derecho)) + 1;

        return y;
    }

    static Nodo insertar(Nodo nodo, int dato) {
        if (nodo == null) return new Nodo(dato);

        if (dato < nodo.dato) {
            nodo.izquierdo = insertar(nodo.izquierdo, dato);
        } else if (dato > nodo.dato) {
            nodo.derecho = insertar(nodo.derecho, dato);
        } else {
            return nodo;
        }

        nodo.altura = Math.max(altura(nodo.izquierdo), altura(nodo.derecho)) + 1;

        int balance = balance(nodo);

        if (balance > 1 && dato < nodo.izquierdo.dato)
            return rotarDerecha(nodo);

        if (balance < -1 && dato > nodo.derecho.dato)
            return rotarIzquierda(nodo);

        if (balance > 1 && dato > nodo.izquierdo.dato) {
            nodo.izquierdo = rotarIzquierda(nodo.izquierdo);
            return rotarDerecha(nodo);
        }

        if (balance < -1 && dato < nodo.derecho.dato) {
            nodo.derecho = rotarDerecha(nodo.derecho);
            return rotarIzquierda(nodo);
        }

        return nodo;
    }

    static void inorden(Nodo nodo) {
        if (nodo != null) {
            inorden(nodo.izquierdo);
            System.out.print(nodo.dato + " ");
            inorden(nodo.derecho);
        }
    }

    public static void main(String[] args) {
        Nodo raiz = null;
        raiz = insertar(raiz, 10);
        raiz = insertar(raiz, 20);
        raiz = insertar(raiz, 30);

        System.out.println("===== ÁRBOL AVL =====");
        System.out.print("Recorrido en inorden: ");
        inorden(raiz);
    }
}

Ejemplo 2: Eliminación en un árbol AVL

Descripción

Se elimina un nodo y el árbol se vuelve a balancear automáticamente.

Pseudocódigo explicativo

INICIO

1. Eliminar como en BST.

2. Actualizar altura.

3. Calcular balance.

4. Aplicar rotaciones si es necesario.

5. Mostrar árbol balanceado.
FIN
Java 

public class Main {
    static class Nodo {
        int dato;
        int altura;
        Nodo izquierdo, derecho;

        Nodo(int dato) {
            this.dato = dato;
            this.altura = 1;
        }
    }

    static int altura(Nodo n) {
        if (n == null) return 0;
        return n.altura;
    }

    static int balance(Nodo n) {
        if (n == null) return 0;
        return altura(n.izquierdo) - altura(n.derecho);
    }

    static Nodo rotarDerecha(Nodo y) {
        Nodo x = y.izquierdo;
        Nodo T2 = x.derecho;
        x.derecho = y;
        y.izquierdo = T2;
        y.altura = Math.max(altura(y.izquierdo), altura(y.derecho)) + 1;
        x.altura = Math.max(altura(x.izquierdo), altura(x.derecho)) + 1;
        return x;
    }

    static Nodo rotarIzquierda(Nodo x) {
        Nodo y = x.derecho;
        Nodo T2 = y.izquierdo;
        y.izquierdo = x;
        x.derecho = T2;
        x.altura = Math.max(altura(x.izquierdo), altura(x.derecho)) + 1;
        y.altura = Math.max(altura(y.izquierdo), altura(y.derecho)) + 1;
        return y;
    }

    static Nodo minimo(Nodo nodo) {
        while (nodo.izquierdo != null) {
            nodo = nodo.izquierdo;
        }
        return nodo;
    }

    static Nodo eliminar(Nodo raiz, int dato) {
        if (raiz == null) return raiz;

        if (dato < raiz.dato) {
            raiz.izquierdo = eliminar(raiz.izquierdo, dato);
        } else if (dato > raiz.dato) {
            raiz.derecho = eliminar(raiz.derecho, dato);
        } else {
            if (raiz.izquierdo == null || raiz.derecho == null) {
                Nodo temp = (raiz.izquierdo != null) ? raiz.izquierdo : raiz.derecho;
                if (temp == null) {
                    raiz = null;
                } else {
                    raiz = temp;
                }
            } else {
                Nodo sucesor = minimo(raiz.derecho);
                raiz.dato = sucesor.dato;
                raiz.derecho = eliminar(raiz.derecho, sucesor.dato);
            }
        }

        if (raiz == null) return raiz;

        raiz.altura = Math.max(altura(raiz.izquierdo), altura(raiz.derecho)) + 1;

        int balance = balance(raiz);

        if (balance > 1 && balance(raiz.izquierdo) >= 0)
            return rotarDerecha(raiz);

        if (balance > 1 && balance(raiz.izquierdo) < 0) {
            raiz.izquierdo = rotarIzquierda(raiz.izquierdo);
            return rotarDerecha(raiz);
        }

        if (balance < -1 && balance(raiz.derecho) <= 0)
            return rotarIzquierda(raiz);

        if (balance < -1 && balance(raiz.derecho) > 0) {
            raiz.derecho = rotarDerecha(raiz.derecho);
            return rotarIzquierda(raiz);
        }

        return raiz;
    }
}
