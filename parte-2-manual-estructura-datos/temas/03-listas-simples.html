Tema 3

Listas enlazadas simples

Fundamentos, nodos, operaciones básicas y aplicaciones de las listas enlazadas simples en programación.

Documentación

Definición

Una lista enlazada simple es una estructura de datos lineal formada por nodos. Cada nodo contiene dos partes: el dato y una referencia al siguiente nodo de la lista. El último nodo apunta a null, lo que indica que no existen más elementos.

Concepto de nodo

Un nodo es la unidad básica de una lista enlazada. Está compuesto por:

  • dato: la información que se almacena.
  • siguiente: la referencia al siguiente nodo de la lista.

Nodo cabeza

El primer nodo de la lista se llama cabeza o head. Desde este nodo se puede recorrer toda la estructura siguiendo las referencias al siguiente elemento.

Características de las listas enlazadas simples

  • No almacenan sus elementos en posiciones contiguas de memoria.
  • Cada nodo apunta únicamente al siguiente.
  • El tamaño puede crecer o disminuir dinámicamente.
  • El acceso a un elemento no es directo como en un array; se debe recorrer la lista.
  • Son útiles cuando se necesitan inserciones y eliminaciones frecuentes.

Ventajas

  • Permiten añadir o eliminar elementos sin mover toda la estructura.
  • Se adaptan bien a cambios en la cantidad de datos.
  • Son más flexibles que los arrays en escenarios dinámicos.

Desventajas

  • El acceso a un elemento específico es más lento que en un array.
  • Requieren memoria adicional para guardar las referencias.
  • El recorrido debe hacerse nodo por nodo.

Operaciones básicas

  • Insertar al inicio
  • Insertar al final
  • Recorrer
  • Buscar
  • Contar elementos

Usos comunes

  • Listas de tareas.
  • Historial de acciones.
  • Gestión de estudiantes.
  • Colecciones que cambian constantemente.
  • Implementación de otras estructuras como pilas y colas.

Ejemplo 1: Registro de estudiantes en una lista enlazada simple

Descripción

Un sistema académico necesita guardar los nombres de varios estudiantes en una lista enlazada simple. Luego debe recorrer la lista completa y mostrar cuántos estudiantes fueron registrados.

Pseudocódigo explicativo

INICIO

1. Crear una lista enlazada simple vacía.
2. Agregar varios nombres de estudiantes al final.
3. Contar la cantidad total de estudiantes.
4. Recorrer la lista desde el primer nodo hasta el final.
5. Mostrar cada nombre con su posición.
6. Mostrar la cantidad total registrada.

FIN
Java 

public class Main {

    // Clase Nodo
    static class Nodo {

        String dato;
        Nodo siguiente;

        Nodo(String dato) {

            this.dato = dato;
            this.siguiente = null;

        }

    }

    // Clase ListaEnlazadaSimple
    static class ListaEnlazadaSimple {

        private Nodo cabeza;

        // Insertar al final
        public void agregarAlFinal(String dato) {

            Nodo nuevo = new Nodo(dato);

            if (cabeza == null) {

                cabeza = nuevo;

            } else {

                Nodo actual = cabeza;

                while (actual.siguiente != null) {

                    actual = actual.siguiente;

                }

                actual.siguiente = nuevo;

            }

        }

        // Contar elementos
        public int contar() {

            int contador = 0;
            Nodo actual = cabeza;

            while (actual != null) {

                contador++;
                actual = actual.siguiente;

            }

            return contador;

        }

        // Recorrer e imprimir
        public void recorrer() {

            Nodo actual = cabeza;
            int posicion = 1;

            while (actual != null) {

                System.out.println(
                    posicion + ". " + actual.dato
                );

                posicion++;
                actual = actual.siguiente;

            }

        }

    }

    public static void main(String[] args) {

        ListaEnlazadaSimple estudiantes =
            new ListaEnlazadaSimple();

        estudiantes.agregarAlFinal("Carlos");
        estudiantes.agregarAlFinal("María");
        estudiantes.agregarAlFinal("Juan");
        estudiantes.agregarAlFinal("Laura");
        estudiantes.agregarAlFinal("Andrés");

        System.out.println(
            "===== REGISTRO DE ESTUDIANTES ====="
        );

        System.out.println(
            "Cantidad de estudiantes: " +
            estudiantes.contar()
        );

        System.out.println("Listado completo:");

        estudiantes.recorrer();

    }

}
Salida esperada 
===== REGISTRO DE ESTUDIANTES =====

Cantidad de estudiantes: 5

Listado completo:

1. Carlos
2. María
3. Juan
4. Laura
5. Andrés

Ejemplo 2: Lista de tareas con prioridad

Descripción

Un sistema de organización de tareas necesita guardar actividades en una lista enlazada simple. La tarea urgente debe insertarse al inicio y las demás al final. Luego se debe mostrar la lista completa y verificar si una tarea específica existe.

Pseudocódigo explicativo

INICIO

1. Crear una lista enlazada simple vacía.
2. Agregar una tarea urgente al inicio.
3. Agregar otras tareas al final.
4. Recorrer la lista y mostrar todas las tareas.
5. Buscar una tarea específica dentro de la lista.
6. Mostrar si la tarea fue encontrada o no.

FIN
Java 

public class Main {

    // Clase Nodo
    static class Nodo {

        String dato;
        Nodo siguiente;

        Nodo(String dato) {

            this.dato = dato;
            this.siguiente = null;

        }

    }

    // Clase ListaEnlazadaSimple
    static class ListaEnlazadaSimple {

        private Nodo cabeza;

        // Insertar al inicio
        public void agregarAlInicio(String dato) {

            Nodo nuevo = new Nodo(dato);

            nuevo.siguiente = cabeza;
            cabeza = nuevo;

        }

        // Insertar al final
        public void agregarAlFinal(String dato) {

            Nodo nuevo = new Nodo(dato);

            if (cabeza == null) {

                cabeza = nuevo;

            } else {

                Nodo actual = cabeza;

                while (actual.siguiente != null) {

                    actual = actual.siguiente;

                }

                actual.siguiente = nuevo;

            }

        }

        // Buscar un dato
        public boolean buscar(String datoBuscado) {

            Nodo actual = cabeza;

            while (actual != null) {

                if (actual.dato.equals(datoBuscado)) {

                    return true;

                }

                actual = actual.siguiente;

            }

            return false;

        }

        // Recorrer e imprimir
        public void recorrer() {

            Nodo actual = cabeza;
            int posicion = 1;

            while (actual != null) {

                System.out.println(
                    posicion + ". " + actual.dato
                );

                posicion++;
                actual = actual.siguiente;

            }

        }

    }

    public static void main(String[] args) {

        ListaEnlazadaSimple tareas =
            new ListaEnlazadaSimple();

        tareas.agregarAlInicio(
            "Atender solicitud urgente"
        );

        tareas.agregarAlFinal(
            "Revisar correo"
        );

        tareas.agregarAlFinal(
            "Actualizar base de datos"
        );

        tareas.agregarAlFinal(
            "Enviar informe"
        );

        String tareaBuscada =
            "Actualizar base de datos";

        System.out.println(
            "===== LISTA DE TAREAS ====="
        );

        System.out.println("Listado completo:");

        tareas.recorrer();

        System.out.println(
            "-----------------------------------"
        );

        System.out.println(
            "Tarea buscada: " + tareaBuscada
        );

        if (tareas.buscar(tareaBuscada)) {

            System.out.println(
                "Resultado: Tarea encontrada"
            );

        } else {

            System.out.println(
                "Resultado: Tarea no encontrada"
            );

        }

    }

}

Listado completo:

1. Atender solicitud urgente
2. Revisar correo
3. Actualizar base de datos
4. Enviar informe

Tarea buscada: Actualizar base de datos

Resultado: Tarea encontrada

Retos

Reto 1: Inventario de contactos

Una agenda digital necesita almacenar varios contactos en una lista enlazada simple. El programa debe mostrar todos los contactos registrados y la cantidad total almacenada.

Pistas

  • Crea una clase para representar cada nodo.
  • El último nodo debe apuntar a null.
  • Recorre la lista desde la cabeza hasta el final.
  • Lleva un contador para saber cuántos contactos hay.
Salida esperada 
===== INVENTARIO DE CONTACTOS =====

Cantidad de contactos: 4

Listado completo:

1. Ana Torres
2. Luis Gómez
3. Camila Pérez
4. Jorge Ruiz

Reto 2: Gestión de prioridades

Un sistema de trabajo recibe tareas normales y una tarea urgente. La tarea urgente debe quedar al inicio de la lista, mientras que las demás deben agregarse al final. Luego se debe mostrar la lista completa y buscar una tarea específica.

Pistas

  • Usa inserción al inicio para la tarea urgente.
  • Usa inserción al final para las tareas normales.
  • Recorre la lista para mostrar el orden final.
  • Implementa una búsqueda sencilla comparando cadenas de texto.
Salida esperada 
===== GESTIÓN DE PRIORIDADES =====

Lista de tareas:

1. Atender emergencia
2. Revisar documentos
3. Preparar informe
4. Enviar correo

Tarea buscada: Preparar informe

Resultado: Tarea encontrada
