Tema 2

Arrays y tipos de datos

Fundamentos de tipos de datos en Java, uso de arrays, organización de información y procesamiento de colecciones de valores mediante estructuras lineales.

Documentación

Definición de tipo de dato

Un tipo de dato indica qué clase de información puede almacenar una variable. En Java, cada variable debe declararse con un tipo específico, lo que ayuda a controlar el valor que se guarda y las operaciones que se pueden realizar con él.

Tipos de datos básicos en Java

int

Almacena números enteros.

double

Almacena números decimales.

char

Almacena un solo carácter.

String

Almacena texto.

boolean

Almacena valores lógicos: true o false.

Definición de array

Un array es una estructura de datos que permite almacenar varios valores del mismo tipo dentro de una sola variable. Cada elemento se guarda en una posición específica llamada índice, y el primer índice siempre es 0.

Características de los arrays

  • Guardan varios datos en una sola estructura.
  • Tienen tamaño fijo una vez creados.
  • Todos sus elementos deben ser del mismo tipo.
  • Se accede a cada elemento mediante su índice.
  • Permiten recorrer los datos de forma ordenada.

Ventajas de usar arrays

  • Organizan información de manera clara.
  • Facilitan el recorrido de datos.
  • Permiten realizar cálculos sobre colecciones de valores.
  • Son útiles para representar listas, registros, calificaciones, inventarios y muchos otros conjuntos de información.

Relación entre tipos de datos y arrays

int[]

Para números enteros.

double[]

Para números decimales.

String[]

Para textos.

boolean[]

Para valores lógicos.

Ejemplo 1: Lista de números y suma total

Descripción

Se tiene un conjunto de números almacenados en un array. El programa debe recorrerlos, mostrarlos en pantalla y calcular la suma total de todos los elementos.

Pseudocódigo explicativo

INICIO

1. Crear un array de números enteros.
2. Inicializar una variable suma en 0.
3. Recorrer el array desde la posición 0 hasta la última.
4. Mostrar cada número.
5. Acumular cada valor en la variable suma.
6. Al final, mostrar la suma total.

FIN
Java 

public class Main {

    public static void main(String[] args) {

        // Ejemplo 1: Lista de números y suma total
        int[] numeros = {10, 20, 30, 40, 50};

        int suma = 0;

        System.out.println(
            "===== LISTA DE NÚMEROS ====="
        );

        for (int i = 0; i < numeros.length; i++) {

            System.out.println(
                "Número: " + numeros[i]
            );

            suma = suma + numeros[i];

        }

        System.out.println();

        System.out.println(
            "Suma total: " + suma
        );

    }
}

Número: 40
Número: 50

Ejemplo 2: Calificaciones de un estudiante

Descripción

Un sistema académico almacena varias calificaciones en un array de tipo decimal. Luego calcula el promedio y determina si el estudiante aprobó o no, según una nota mínima de 3.0.

Pseudocódigo explicativo

INICIO

1. Crear un array con calificaciones decimales.
2. Inicializar una variable suma en 0.
3. Recorrer el array y sumar cada calificación.
4. Calcular el promedio dividiendo la suma entre la cantidad de notas.
5. Mostrar cada calificación.
6. Mostrar el promedio.
7. Si el promedio es mayor o igual a 3.0, mostrar "Aprobado".
8. Si no, mostrar "Reprobado".

FIN
Java 

public class Main {

    public static void main(String[] args) {

        // 1. Crear el array con calificaciones decimales
        double[] calificaciones = {
            4.5,
            3.8,
            4.0,
            2.5,
            3.2
        };

        double suma = 0;

        System.out.println(
            "===== CALIFICACIONES ====="
        );

        // 2. Recorrer el array y sumar cada calificación
        for (int i = 0; i < calificaciones.length; i++) {

            System.out.println(
                "Nota " + (i + 1) + ": " +
                calificaciones[i]
            );

            suma = suma + calificaciones[i];

        }

        // 3. Calcular el promedio
        double promedio =
            suma / calificaciones.length;

        System.out.println(
            "---------------------------"
        );

        System.out.println(
            "Promedio: " + promedio
        );

        // 4. Determinar si aprobó o reprobó
        if (promedio >= 3.0) {

            System.out.println(
                "Estado: Aprobado"
            );

        } else {

            System.out.println(
                "Estado: Reprobado"
            );

        }

    }
}
Salida esperada 
===== CALIFICACIONES =====

Nota 1: 4.5
Nota 2: 3.8
Nota 3: 4.0
Nota 4: 2.5
Nota 5: 3.2

Promedio: 3.6

Estado: Aprobado

Reto 1: Número mayor en un array

Dado un conjunto de números almacenados en un array, se debe identificar cuál es el valor mayor y mostrarlo en pantalla.

Pistas

  • Usa una variable llamada mayor.
  • Inicialízala con el primer valor del array.
  • Recorre el arreglo y compara cada elemento.
  • Si un valor es mayor, actualiza la variable.
Salida esperada 
Números: 15, 8, 22, 5, 18

Número mayor: 22

Reto 2: Contar números pares

Dado un array de números enteros, se debe contar cuántos de ellos son pares y mostrar el resultado.

Pistas

  • Usa el operador módulo %.
  • Un número es par si numero % 2 == 0.
  • Lleva un contador para registrar cuántos pares encuentras.
  • Recorre todo el array con un ciclo.
Salida esperada 
Números: 12, 7, 4, 9, 2, 15

Cantidad de números pares: 3
