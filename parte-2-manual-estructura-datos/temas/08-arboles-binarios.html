Tema 8

Árboles binarios

Estructuras jerárquicas, recorridos, operaciones y aplicaciones de árboles binarios.

Documentación

Definición de árbol binario

Un árbol binario es una estructura de datos jerárquica en la que cada nodo puede tener como máximo dos hijos: un hijo izquierdo y un hijo derecho. Se utiliza para organizar información de forma que facilite la búsqueda y manipulación de datos.

Estructura de un árbol binario

  • Nodo raíz: Primer nodo del árbol.
  • Nodo padre: Nodo que tiene hijos.
  • Nodo hijo: Nodo que depende de otro.
  • Nodo hoja: Nodo sin hijos.
  • Subárbol: Cada hijo puede ser raíz de otro árbol.

Tipos de recorridos en árboles binarios

  • Preorden (Raíz - Izquierda - Derecha)
  • Inorden (Izquierda - Raíz - Derecha)
  • Postorden (Izquierda - Derecha - Raíz)

Características de los árboles binarios

  • Cada nodo tiene máximo dos hijos.
  • Estructura jerárquica.
  • Permite búsquedas eficientes.
  • Puede crecer dinámicamente.

Ventajas de los árboles binarios

  • Organización eficiente de datos.
  • Permite búsquedas rápidas (especialmente en árboles balanceados).
  • Flexible para representar relaciones jerárquicas.

Desventajas de los árboles binarios

  • Puede volverse ineficiente si no está balanceado.
  • Requiere más memoria que estructuras lineales.
  • Implementación más compleja.

Usos comunes

  • Representación de estructuras jerárquicas.
  • Árboles de búsqueda (BST).
  • Expresiones matemáticas.
  • Sistemas de archivos.
  • Inteligencia artificial y juegos.

Ejemplo 1: Creación y recorrido de un árbol binario

Descripción

Un sistema crea un árbol binario con valores numéricos y luego muestra los recorridos en preorden, inorden y postorden.

Pseudocódigo explicativo 

INICIO

1. Crear nodos del árbol.

2. Asignar hijos izquierdo y derecho.

3. Recorrer el árbol en preorden.

4. Recorrer el árbol en inorden.

5. Recorrer el árbol en postorden.

6. Mostrar resultados.

FIN
Java 

public class Main {

    static class Nodo {

        int dato;
        Nodo izquierdo;
        Nodo derecho;

        Nodo(int dato) {

            this.dato = dato;
            this.izquierdo = null;
            this.derecho = null;

        }

    }

    // Recorrido preorden
    public static void preorden(Nodo nodo) {

        if (nodo != null) {

            System.out.print(nodo.dato + " ");
            preorden(nodo.izquierdo);
            preorden(nodo.derecho);

        }

    }

    // Recorrido inorden
    public static void inorden(Nodo nodo) {

        if (nodo != null) {

            inorden(nodo.izquierdo);
            System.out.print(nodo.dato + " ");
            inorden(nodo.derecho);

        }

    }

    // Recorrido postorden
    public static void postorden(Nodo nodo) {

        if (nodo != null) {

            postorden(nodo.izquierdo);
            postorden(nodo.derecho);
            System.out.print(nodo.dato + " ");

        }

    }

    public static void main(String[] args) {

        // Crear árbol
        Nodo raiz = new Nodo(10);

        raiz.izquierdo = new Nodo(5);
        raiz.derecho = new Nodo(15);

        raiz.izquierdo.izquierdo = new Nodo(3);
        raiz.izquierdo.derecho = new Nodo(7);

        System.out.println(
            "===== ÁRBOL BINARIO ====="
        );

        System.out.print("Preorden: ");
        preorden(raiz);

        System.out.println();

        System.out.print("Inorden: ");
        inorden(raiz);

        System.out.println();

        System.out.print("Postorden: ");
        postorden(raiz);

    }

}
Salida esperada 
===== ÁRBOL BINARIO =====

Preorden:
10 5 3 7 15

Inorden:
3 5 7 10 15

Postorden:
3 7 5 15 10

Ejemplo 2: Cálculo de la suma de nodos en un árbol

Descripción

Un sistema necesita calcular la suma total de todos los valores almacenados en un árbol binario utilizando recursividad.

Pseudocódigo explicativo 

INICIO

1. Crear un árbol binario.

2. Definir una función recursiva:

a. Si el nodo es nulo, retornar 0.

b. Retornar valor del nodo +
   suma izquierda +
   suma derecha.

3. Mostrar el resultado total.

FIN
Java 

public class Main {

    static class Nodo {

        int dato;
        Nodo izquierdo;
        Nodo derecho;

        Nodo(int dato) {

            this.dato = dato;

        }

    }

    // Función para sumar nodos
    public static int sumar(Nodo nodo) {

        if (nodo == null) {

            return 0;

        }

        return nodo.dato +
               sumar(nodo.izquierdo) +
               sumar(nodo.derecho);

    }

    public static void main(String[] args) {

        Nodo raiz = new Nodo(10);

        raiz.izquierdo = new Nodo(5);
        raiz.derecho = new Nodo(15);

        raiz.izquierdo.izquierdo = new Nodo(3);
        raiz.izquierdo.derecho = new Nodo(7);

        System.out.println(
            "===== SUMA DE NODOS ====="
        );

        int total = sumar(raiz);

        System.out.println(
            "Suma total: " + total
        );

    }

}
Salida esperada 
===== SUMA DE NODOS =====

Suma total: 40

Retos

Reto 1: Contar nodos de un árbol

Crear un programa que cuente la cantidad total de nodos en un árbol binario.

Pistas

  • Usa recursividad.
  • Cuenta el nodo actual + nodos izquierdos + nodos derechos.
  • Caso base: nodo nulo.

Reto 2: Altura de un árbol

Determinar la altura de un árbol binario (la longitud del camino más largo desde la raíz hasta una hoja).

Pistas

  • Usa recursividad.
  • Altura = 1 + máximo entre altura izquierda y derecha.
  • Caso base: árbol vacío.

Altura: 3
